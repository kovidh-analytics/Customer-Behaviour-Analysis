# Customer-Behaviour-Analysis

This project performs an end-to-end data analytics workflow to understand customer shopping behaviour and generate actionable business insights.
It covers data loading and cleaning in Python, SQL-based analysis using relational databases, interactive dashboard creation in Power BI, and final reporting through documents and presentations.
